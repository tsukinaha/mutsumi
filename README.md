### Mutsumi

#### Mutsumi Lib
lib for GTK4 -> MPV Video + Danmaku + Control
Support gpu-next/dmabuf-wayland and some vo, embed by wl-proxy.

#### Mutsumi Live
Support some live with danmaku
**DO NOT PACKAGE THIS**

#### Mutsumi Player (WIP)
General video player

`build.rs` will automatically detect the blp files and convert them to ui xml, and generate the gresource file.
